# upship-web Project

Practice WebBoard(Upship-Web)

1. 유형 : 게시판

2. 개발스택 :
   1) Web-Server : Apache Tom-cat 
   2) IDE : Eclipse
   3) DataBase : Oracle SQL Developer
   4) Back-End
      - Java, Oracle SQL
   5) Front-End
      - HTML5, CSS3, JavaScript, Jquery, Ajax, Bootstrap
      
3. 주요 기능
   : 로그인, 회원가입, 게시물 등록/조회/수정/삭제, 좋아요 기능, 댓글 기능, 검색, 정렬, 파일업로드, 자료실, 개인 게시판(단, 회원에 한해서)
